Core Features

Everything a recruiter needs, built in

Recruiter Hiring Manager
📄

Resume & JD Upload

Bulk upload resumes (PDF/DOCX) and paste or upload job descriptions. One job, many candidates — instantly organized.

🤖

AI Candidate Analysis

Match score, strengths, weaknesses, missing skills, and a recommendation — generated per candidate against the job.

🏆

Candidate Ranking

All applicants sorted by AI fit score. Top-recommended candidates highlighted so the best rises to the top automatically.

🗂️

Hiring Pipeline

Kanban pipeline: Applied → Screened → Shortlisted → Interview → Offer → Hired. Every candidate has a clear status.

📊

Skill Gap Report

Visual breakdown of skills most commonly missing across the applicant pool — helps refine job requirements over time.

🖥️

Recruiter Dashboard

Overview of open roles, total applicants, screening progress, and shortlist counts — at a glance, always current.
