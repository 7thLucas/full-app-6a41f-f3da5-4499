# RecruitIQ — Product Overview

## Brand
- **Name:** RecruitIQ
- **Tagline:** Screen smarter. Hire faster.
- **Voice:** Professional, data-driven, not robotic

## Users
- **Primary:** Recruiters and Hiring Managers
- **Workspace scope:** Single org workspace (MVP)

## Core Features

### Resume Screening
- Upload resumes (PDF/DOCX) and job descriptions
- AI match scoring: 0–100% per candidate vs job requirements
- Output per candidate: strengths, weaknesses, missing skills, hire recommendation
- Candidate ranking by overall fit score

### Kanban Pipeline
Stages: Applied → Screened → Shortlisted → Interview → Offer → Hired / Rejected
- Drag-and-drop candidate cards across pipeline stages
- Visual status tracking per role

### Recruiter Dashboard
- Open roles overview with applicant counts
- Screening progress indicators per role
- Quick-access to top-ranked candidates

### Skill Gap Report
- Aggregate skill gap analysis across the full applicant pool
- Identify most common missing skills per role

## MVP Scope
- Single org workspace
- No ATS, email, or calendar integrations
- Core flow: upload → score → rank → pipeline

## Strategic Principles
- Speed and clarity over complexity
- AI output must be explainable, not a black box
- Recruiters trust data; every number must mean something
